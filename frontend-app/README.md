cd # MMX Nexus Platform

A premium Marketing Mix Modeling (MMM) interface.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Premium CSS (Dark Mode, Glassmorphism)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Motion**: Framer Motion

## Getting Started

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Start Development Server**

   ```bash
   npm run dev
   ```

## Integration

This frontend is designed to consume the MMX Agent Backend API (Flask) running at `http://localhost:8080`.
Update `vite.config.js` to proxy requests if needed.
